
# 学习通选择题复习工具

从超星学习通（chaoxing）已批改的作业回看页面提取选择题，在本地浏览器中刷题复习。

## 设计初衷

很多同学第一次做学习通作业时，会用脚本或 AI 帮忙完成，作业虽然交了但对知识其实没掌握。这个工具就是针对这种情况——作业批改后，**回到已显示正确答案的回看页面，把题目、选项和正确答案一并爬下来**，存成本地 JSON，然后在复习页面里反复刷题，直到真正记住。

> 期末想复习？把之前作业的题目再抓一遍，导入就能刷，不用重新翻网页。

## 项目文件

| 文件 | 说明 |
|------|------|
| `cx_question_scraper.user.js` | Tampermonkey 用户脚本，在已批改的学习通回看页面添加「导出选择题」按钮，提取题目、选项和正确答案，导出为 JSON |
| `quiz.html` | 纯前端复习页面，支持导入 JSON 后逐题练习，含错题本、进度统计、快速跳转 |
| `course_name/` | 各章节题库 JSON（随 gitignore，不公开，需自行导出） |

## 使用流程
### 1. 安装脚本

在 Edge/Chrome 中安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展，然后新建脚本，把 `cx_question_scraper.user.js` 的内容粘贴进去保存。

### 2. 导出题目（关键步骤）

- 打开学习通的**作业回看页面**（必须是已经批改、显示了正确答案的页面！）
- 如果脚本已启用，页面右上角会出现「导出选择题」按钮
- 点击按钮，在弹窗中可以复制 JSON 或直接下载文件
- **注意**：页面刷新后按钮没出现，请确认 Tampermonkey 已启用，且当前页面是批改后的回看页（不是未提交的作业页）

> 什么时候用？作业发下来做完了 → 老师批改/自动批改完成 → 回到作业回看页面（能看到正确答案）→ 点导出。

### 3. 本地复习

用浏览器打开 `quiz.html`，点击「选择文件」导入导出的 JSON 文件，即可开始答题。

功能：
- 单选题 / 多选题
- 答错自动加入错题本
- 可切换「全部题目」和「错题」复习模式
- 进度条显示掌握程度
- 快速跳转到指定题号


## 隐私说明

- 所有数据仅存储在本地浏览器中，不上传任何服务器
- 导出的 JSON 仅包含题目、选项、答案，不含个人身份信息
- 题库文件 course_name/ 默认不纳入版本控制
